# 奇语言（Qi）v0.1.3-beta

用中文写代码，像写文章一样自然。

## 快速开始

```bash
# 运行文件
python3 -m qi run 示例/你好.qi
# 或
python3 -m qi 运行 示例/你好.qi

# 交互式 Shell
python3 -m qi shell

# 格式化代码
python3 -m qi fmt 文件.qi
python3 -m qi 格式化 -i 文件.qi   # -i 直接修改文件
```

> 也支持旧入口：`python3 奇.py 文件.qi`

## 语法示例

```
令 整数 X 为 10。令 文本 名字 为 "奇语言"。
输出 X 加 5。                    // 15

若 X 大于 0 则：输出 "正"。否则：输出 "负"。    // 条件
当 X 小于 3 时：设 X 为 X 加 1；输出 X。       // 循环

令 整数 求和 为（整数 甲、整数 乙）：返回 甲 加 乙。  // 函数
令 整数列 列表 为 [1、2、3]。                      // 列表
令 结构 学生 含：文本 姓名、整数 年龄。             // 结构体
设 小明的年龄 为 19。                               // 成员修改
令 Y 为 若 X 大于 0 则 "正" 否则 "负"。             // 三元表达式

输出 文本库的长度("hello")。                        // 标准库
```

## 核心特性

| 特性 | 说明 |
|------|------|
| 上下文关键字 | 关键字可当变量名（`长度`、`文本库` 不再冲突） |
| 静态类型 | `令 整数 X 为 10` |
| 中文运算符 | `加`、`减`、`乘`、`除`、`且`、`或`、`抑或` |
| 块结构 | `：` 开始、`。` 结束、`；` 连接多语句 |
| 字节码 VM | 栈式虚拟机（`交互式奇解析器.py` 默认启用） |
| 标准库 | 基础库、文本库、列表库、文件库 |
| 结构体 | `令 结构 学生 含：文本 姓名、整数 年龄` |
| 列表 | `令 整数列 X 为 [1、2、3]`（1-based 索引） |
| 函数 | 递归支持、函数作为值传递 |
| 三元表达式 | `若 条件 则 A 否则 B` |
| 文件后缀 | `.qi`/`.奇`（源码）、`.qih`/`.奇头`（头文件） |

## 标准库

| 模块 | 功能 |
|------|------|
| 基础库 | `数值化`、`文字化`、`取模`、`最大值`、`最小值` |
| 文本库 | `长度`、`截取`、`查找`、`替换`、`转大写`、`转小写`、`去留白` |
| 列表库 | `排序`、`反转`、`去重`、`拥有`、`长度` |
| 文件库 | `打开`、`读取`、`写入`、`关闭`（VFS 抽象） |

```
文本库的长度("hello")     → 5
基础库的数值化("123")      → 123
列表库的排序([3、1、2])    → [1、2、3]
```
