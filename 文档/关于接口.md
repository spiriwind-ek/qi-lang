# 奇语言 API 参考

## 运行方式

```bash
# 子命令模式（推荐）
python3 -m qi run 文件.qi
python3 -m qi 运行 文件.qi

# 格式化
python3 -m qi fmt 文件.qi
python3 -m qi 格式化 -i 文件.qi

# 交互式 Shell
python3 -m qi shell

# 旧入口（仍可用）
python3 奇.py 文件.qi
```

## 词法分析器 (Lexer)

### `Lexer(source: str)`

**参数：** 源代码字符串

**方法：** `tokenize() -> list[Token]`

**示例：**
```python
from 词法分析器 import Lexer

tokens = Lexer('令 整数 X 为 10。').tokenize()
for token in tokens:
    print(token)
```

---

## 语法分析器 (Parser)

### `Parser(tokens: list[Token])`

**参数：** token 列表

**方法：** `parse() -> Program`

**示例：**
```python
from 词法分析器 import Lexer
from 语法分析器 import Parser

tokens = Lexer('令 整数 X 为 10。').tokenize()
ast = Parser(tokens).parse()
```

---

## 解释器 (Interpreter)

### `Interpreter(use_vm: bool = True)`

创建解释器实例。默认启用字节码 VM（`use_vm=True`）。

**方法：** `run(source: str) -> list[str]`

**示例：**
```python
from 解释器 import Interpreter

interp = Interpreter(use_vm=True)
result = interp.run('令 整数 X 为 10。输出 X。')
print(result)  # ['10']
```

---

## 编译器 (Compiler)

### `Compiler(structs=None, enclosing=None)`

**方法：** `compile(node) -> Chunk`

将 AST 编译为字节码 Chunk。

**示例：**
```python
from 词法分析器 import Lexer
from 语法分析器 import Parser
from 编译器 import Compiler

tokens = Lexer('令 整数 X 为 10。输出 X。').tokenize()
ast = Parser(tokens).parse()
compiler = Compiler()
chunk = compiler.compile(ast)
```

---

## 虚拟机 (VM)

### `VM()`

**方法：** `run(chunk: Chunk)`

执行字节码 Chunk。

**示例：**
```python
from 词法分析器 import Lexer
from 语法分析器 import Parser
from 编译器 import Compiler
from 虚拟机 import VM

tokens = Lexer('令 整数 X 为 10。输出 X。').tokenize()
ast = Parser(tokens).parse()
chunk = Compiler().compile(ast)
vm = VM()
vm.run(chunk)
print(vm.get_output())  # ['10']
```

## 格式化器 (Formatter)

### `format_qi(raw: str) -> str`

格式化奇语言代码字符串。

**示例：**
```python
from 格式化奇 import format_qi

formatted = format_qi('令整数X为10。输出X。')
print(formatted)
# 令 整数 X 为 10。
# 输出 X。
```

## 环境 (Environment)

### `Environment(parent: Environment = None)`

运行时环境，管理变量、函数、结构体、模块。

**方法：**
- `declare_var(name, value, type_name)` — 声明变量
- `assign_var(name, value)` — 修改变量
- `get_var(name)` — 读取变量
- `declare_func(name, node)` — 声明函数
- `get_func(name)` — 获取函数
- `_infer_type(value)` — 从值推断类型
