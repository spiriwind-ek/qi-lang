# 常见问题

## 运行相关

### 怎么运行奇语言代码？

```bash
python3 -m qi run 文件.qi
python3 -m qi 运行 文件.qi
```

### 怎么格式化代码？

```bash
python3 -m qi fmt -i 文件.qi
```

### 交互式 Shell 怎么用？

```bash
python3 -m qi shell
qi> 输出 "你好"。
qi> 退出
```

### Windows 怎么运行？

Windows 上 Python 是 `py` 命令：

```bash
py -m qi run 文件.qi
```

## 语法相关

### 注释怎么写？

```
注：这是注释。
```

### 变量怎么声明？

```
令 整数 X 为 10。
```

### 字符串用什么引号？

中文引号 `""`：

```
令 文本 S 为 "你好"。
```

### 怎么读取用户输入？

```
读（"请输入："）。
输出 _input。
```

## 错误相关

### "变量未定义"

```
令 整数 X 为 10。
输出 Y。  ← 错误！Y 未定义
```

### "变量已存在"

```
令 整数 X 为 10。
令 整数 X 为 20。  ← 错误！用 设 修改
```

### 格式错误

检查 `：`/`。` 是否配对。可以用格式化工具自动修复：

```bash
python3 -m qi fmt -i 文件.qi
```

## 技术相关

### 奇语言是用什么写的？

Python。编译器、虚拟机、格式化工具全是 Python。

### 为什么叫"奇"？

因为中文编程语言很"奇"特（也谐音 Qi）。

### 奇语言和文语是什么关系？

奇语言是 Python 原型。文语（wen-lang）是同一设计思路的 C 语言实现，目标是裸机运行。

### 编译缓存（.简令）是什么？

第一次运行后，目录里会出现 `_qicache_/`，里面是编译后的字节码文件。下次运行若源码没改过，直接加载缓存。
