## 入门

### 运行方式

```bash
# 运行文件
python3 -m qi run hello.qi
# 或中文子命令
python3 -m qi 运行 hello.qi

# 格式化代码
python3 -m qi fmt -i hello.qi

# 交互式 Shell
python3 -m qi shell
qi> 输出 "你好，世界！"。
你好，世界！
qi> 退出
```

### 第一个程序

```
注：这是我的第一个奇语言程序
输出 "你好，世界！"。
```

保存为 `hello.qi`，运行：

```bash
python3 -m qi run hello.qi
```

### 编译缓存

第一次运行后，目录里会出现 `_qicache_/` 文件夹，里面是编译产物（`.简令` 文件）。下次运行若源码没改过，直接加载缓存，跳过编译。

## 变量

### 声明

```
令 整数 年龄 为 18。
令 文本 名字 为 "张三"。
令 小数 身高 为 1.75。
令 布尔 已婚 为 假。
```

`令` 声明一个变量，格式：`令 类型 名字 为 值。`

### 类型推导

```
令 X 为 10。      注：自动推断为整数
令 Y 为 "hello"。 注：自动推断为文本
```

### 修改

```
令 整数 X 为 10。
设 X 为 20。
设 X 为 X 加 1。
```

## 运算

### 算术

```
令 整数 X 为 10。
令 整数 Y 为 3。
输出 X 加 Y。    注：13
输出 X 减 Y。    注：7
输出 X 乘 Y。    注：30
输出 X 除 Y。    注：3
```

### 比较与逻辑

```
输出 X 等于 10。      注：真
输出 X 大于 5。       注：真
输出 真 且 假。       注：假
输出 真 或 假。       注：真
输出 非 真。          注：假
```

### 中文数字

源码里可以直接用中文数字：

```
令 整数 甲 为 二十。
输出 甲。  注：20
```

支持：一、十、十五、二十五、一百、一百二十三……

## 控制结构

`：`/`。` 括号式配对。块内多条语句用 `；` 连接。

### 条件

```
若 X 大于 0 则：输出 "正数"。否则：输出 "非正数"。
```

多语句块：

```
若 X 大于 0 则：令 整数 Y 为 1；设 Y 为 Y 加 1。否则：输出 "负数"。
```

多分支：

```
若 X 大于 0 则：输出 "正"。再若 X 等于 0 则：输出 "零"。否则：输出 "负"。
```

### 条件循环

```
令 整数 X 为 0。
当 X 小于 10 时：设 X 为 X 加 1。
输出 X。
```

### 计数循环

```
重复 3 次：输出 "好"。  注：输出好、好、好
```

### 遍历循环

```
令 整数列 水果 为 ["苹果"、"香蕉"、"橙子"]。
对 水果 的 每个 名称：输出 名称。
```

## 函数

### 定义与调用

```
令 整数 求和 为（整数 甲、整数 乙）：返回 甲 加 乙。
输出 求和（3、5）。  注：8
```

函数体支持所有语句类型——if、while、repeat、foreach、input 都可以写在函数里：

```
令 空 问候 为（整数 n）：
    重复 n 次：输出 "你好"。
问候（3）。
```

### 递归

```
注：计算 5 的阶乘
令 整数 阶乘 为（整数 n）：
    若 n 等于 1 则：返回 1。
    返回 n 乘 阶乘（n 减 1）。
输出 阶乘（5）。  注：120
```

## 数据结构

### 列表

```
令 整数列 库存 为 [10、20、30]。
输出 库存[1]。      注：10（索引从 1 开始）
输出 库存的长度。    注：3
```

列表操作：

```
令 整数列 数据 为 [3、1、2]。
调 列表库的排序（数据）。
输出 数据[1]。  注：1
```

### 结构体

```
令 结构 学生 含：文本 姓名、整数 年龄。
令 学生 小明 为 学生含（"小明"、18）。
输出 小明的姓名。    注：小明
设 小明的年龄 为 19。
```

### 成员方法调用

```
输出 文本库的长度（"hello"）。  注：5
输出 文本库的替换（"a-b-c"、"-"、"/"）。  注：a/b/c
```

## 代码格式化

```
python3 -m qi fmt -i hello.qi
```

`-i` 直接修改文件。不加 `-i` 则在终端显示格式化结果。

格式化工具会自动添加空格、转换中文数字、统一标点。

## 最佳实践

### 命名

变量名、函数名用中文，清晰表意：

```
令 整数 用户年龄 为 18。
令 整数 计算平方 为（整数 X）：返回 X 乘 X。
```

### 注释

```
注：这个函数计算两个数的和
令 整数 求和 为（整数 甲、整数 乙）：返回 甲 加 乙。
```

### 善用格式化

写代码时可以偷懒不加空格：

```
令整数甲为十。输出甲。
```

然后运行格式化：

```bash
python3 -m qi fmt -i hello.qi
```

自动变成：

```
令 整数 甲 为 10。
输出 甲。
```
