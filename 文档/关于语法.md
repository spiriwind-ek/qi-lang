# 奇语言 语法规范

## 设计哲学

用中文写代码，像写文章一样自然。

## 基本约定

1. 语句以 `。` 结尾
2. 代码块用 `：` 开始，**缩进**结束（类似 Python）
3. 块内多条语句换行写，自动缩进对齐
4. 同一行内多条语句用 `；` 连接
5. 参数用 `、` 分隔
6. 变量声明必须指定类型，声明后类型不可变更（支持类型推导）
7. 成员访问用 `的`

## 标点符号

| 符号 | 用途 |
|------|------|
| `。` | 语句结束 |
| `；` | 行内连接多条语句 |
| `：` | 代码块开始 |
| `、` | 分隔参数 |
| `（）` | 参数列表 / 函数调用 / 输入提示 |
| `"` | 文本字面量 |
| `[` `]` | 列表字面量 / 列表索引 |

## 类型系统

### 基础类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `整数` | 整数 | `令 整数 X 为 42。` |
| `小数` | 浮点数 | `令 小数 X 为 3.14。` |
| `文本` | 字符串 | `令 文本 S 为 "你好"。` |
| `布尔` | 布尔值 | `令 布尔 B 为 真。` |
| `空` | 函数无返回值 | `令 空 F 为（）：输出 1。` |
| `无` | 空值 | `输出 无。` |

### 复合类型

| 类型 | 声明方式 | 说明 |
|------|----------|------|
| `整数列` | `令 整数列 X 为 [1、2、3]。` | 整数列表 |
| `小数列` | `令 小数列 X 为 [1.1、2.2]。` | 浮点列表 |
| `文本列` | `令 文本列 X 为 ["a"、"b"]。` | 字符串列表 |
| `结构` | `令 结构 学生 含：文本 姓名。` | 结构体 |

## 控制结构

缩进分块——`：` 后换行 + 缩进，缩进归零自动结束块。

### 条件

```qi
若 X 大于 0 则：
    输出 "正数"。
否则：
    输出 "非正数"。
```

多分支（`再若`）：

```qi
若 X 大于 0 则：
    输出 "正数"。
再若 X 等于 0 则：
    输出 "零"。
否则：
    输出 "负数"。
```

行内单语句（兼容旧风格）：

```qi
若 X 大于 0 则：输出 "正数"。否则：输出 "非正数"。
```

### 条件循环

```qi
当 X 小于 10 时：
    设 X 为 X 加 1。
```

行内：

```qi
当 X 小于 3 时：设 X 为 X 加 1；输出 X。
```

### 计数循环

```qi
重复 5 次：
    输出 "Hello"。
```

### 遍历循环

```qi
令 整数列 水果 为 ["苹果"、"香蕉"、"橙子"]。
对 水果 每项 名称 于：
    输出 名称。
```

## 函数

### 定义

缩进模式（推荐）：

```qi
令 整数 求和 为（整数 甲、整数 乙）：
    返回 甲 加 乙。
```

行内单语句（兼容旧风格）：

```qi
令 整数 求和 为（整数 甲、整数 乙）：返回 甲 加 乙。
```

函数体内支持所有语句类型：

```qi
令 空 问候 为（整数 n）：
    重复 n 次：
        输出 "你好"。
```

### 递归

```qi
令 整数 阶乘 为（整数 n）：
    若 n 等于 1 则：
        返回 1。
    否则：
        返回 n 乘 阶乘（n 减 1）。
```

### 调用

```
令 整数 结果 为 求和（3、5）。
输出 结果。
```

## 列表

```
令 整数列 库存 为 [10、20、30]。
输出 库存[1]。        注：10（索引从 1 开始）
输出 库存的长度。      注：3
设 库存[2] 为 99。    注：修改
```

## 结构体

```
令 结构 学生 含：文本 姓名、整数 年龄。
令 学生 小明 为 学生含（"小明"、18）。
输出 小明的姓名。    注：小明
设 小明的年龄 为 19。
```

## 成员方法调用

```
调 列表库的排序（数据）。
输出 文本库的长度（"hello"）。    注：5
输出 文本库的替换（"a-b-c"、"-"、"/"）。  注：a/b/c
```

`调` 用于将方法调用作为独立语句执行（不返回值）。

## 输入输出

```
读（"请输入你的名字："）。
输出 "你好，" 加 _input。
```

`读` 读取用户输入，结果存入 `_input` 变量。

## 模块导入

```
包括 "./数学工具.qi"。
```

`包括` 在编译时加载并编译指定文件。不支持绝对路径。

## 注释

```
注：这是注释，不会被执行。
```

## 上下文关键字

奇语言的关键字是**上下文相关**的。同一词在不同位置可以是关键字或变量名：

```
令 整数 长度 为 10。        注：长度 是变量名
输出 列表的长度。             注：长度 是关键字（在 的 后面）
文本库                        注：模块名（不会被拆成 文本+库）
文本库的长度("hello")        注：标准库方法调用
```

## 标准库

通过 `模块.方法(参数)` 调用：

| 调用 | 结果 |
|------|------|
| `文本库的长度("hello")` | 5 |
| `基础库的数值化("123")` | 123 |
| `列表库的排序([3、1、2])` | [1、2、3] |
| `列表库的拥有([1、2、3]、2)` | 真 |

详见[关于接口](关于接口.md)。

## 代码格式化

```
python3 -m qi fmt -i 文件.qi
```

格式化工具自动添加空格、转换中文数字、统一标点。

## 编译缓存

运行后自动生成 `_qicache_/` 目录，存放编译后的 `.简令`（`.qic`）文件。下次运行若源码没改过，直接加载缓存。
