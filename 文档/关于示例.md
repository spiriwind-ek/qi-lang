# 奇语言 示例

奇语言的示例代码放在 `示例/` 目录下。

## 运行示例

```bash
python3 -m qi run 示例/你好.qi
python3 -m qi 运行 示例/斐波那契.qi
```

## 示例列表

| 文件 | 说明 |
|------|------|
| `你好.qi` | Hello World |
| `变量.qi` | 变量声明与修改 |
| `数学.qi` | 算术运算 |
| `循环.qi` | 循环结构（当、重复、遍历） |
| `斐波那契.qi` | 递归函数 |
| `函数式.qi` | 函数定义与调用 |
| `二叉树.qi` | 结构体与递归 |
| `结构体二叉树.qi` | 结构体嵌套 |
| `排序.qi` | 列表排序 |
| `数组.qi` | 列表操作 |
| `立方体.qi` | 结构体实例化 |
| `测试导入.qi` | 模块导入 |
| `枚举表达式.qi` | 枚举/条件表达式 |

## 运行所有示例

```bash
for f in 示例/*.qi; do
    echo "=== $f ==="
    python3 -m qi run "$f"
done
```
