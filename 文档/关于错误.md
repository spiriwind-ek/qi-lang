# 错误信息参考

## 运行时错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| `变量 'X' 未定义` | 使用了未声明的变量 | 先用 `令` 声明变量 |
| `变量 'X' 已存在` | 重复声明已存在的变量 | 用 `设` 修改变量 |
| `类型不匹配：期望 整数，得到 文本` | 赋值类型与声明类型不一致 | 检查类型是否匹配 |
| `除零错误` | 除数为零 | 检查除数是否可能为零 |
| `函数 'X' 未定义` | 调用了未定义的函数 | 先定义函数再调用 |
| `参数数量不匹配` | 函数调用参数数量不对 | 检查函数定义和调用参数数 |
| `索引超出范围` | 列表索引越界 | 列表索引从 1 开始 |
| `不是列表` | 对非列表类型做列表操作 | 检查变量是否为列表 |
| `不是结构体` | 对非结构体做结构体操作 | 检查变量是否为结构体 |
| `结构体成员 'X' 未定义` | 访问了不存在的成员 | 检查结构体定义中的字段名 |
| `只能调用函数` | 尝试调用非函数值 | 检查调用的对象 |
| `疑似无限循环` | 循环超过 100000 次 | 检查循环条件是否可能永远为真 |
| `文件不存在` | Include 的文件不存在 | 检查文件路径 |

## 语法错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| `期望 ...，得到 ...` | 语法不匹配 | 检查语句格式 |
| `不支持的语句` | 无法识别的语句开头 | 检查关键字拼写 |
| `意外的表达式` | 表达式位置不正确 | 检查表达式结构 |

## 兼容性

| 问题 | 说明 |
|------|------|
| 半角 `,` | 请用中文 `、` 分隔参数 |
| 半角 `;` | 块内连接多条语句用 `；` |
| 半角 `:` | 代码块开始用 `：` |
| 半角 `.` | 语句结束用 `。` |
