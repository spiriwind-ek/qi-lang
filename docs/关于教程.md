## 控制结构

`:``。` 括号式配对。块内多条语句用 `；` 连接。

### 条件

```
若 X 大于 0 则：输出 "正数"。否则：输出 "非正数"。
```

多语句块：

```
若 X 大于 0 则：令 整数 Y 为 1；设 Y 为 Y 加 1。否则：输出 "负数"。
```

多分支：

```
若 X 大于 0 则：输出 "正"。再若 X 等于 0 则：输出 "零"。否则：输出 "负"。
```

### 循环

```
令 整数 X 为 0。
当 X 小于 10 时：设 X 为 X 加 1。
输出 X。
```

多语句循环：

```
令 整数 X 为 0。
当 X 小于 3 时：设 X 为 X 加 1；输出 X。
```

### 遍历

```
令 整数列 水果 为 ["苹果"、"香蕉"、"橙子"]。
对 水果 每项 名称 于：输出 名称。
```

## 函数

### 定义与调用

```
令 整数 求和 为（整数 甲、整数 乙）：返回 甲 加 乙。
令 整数 结果 为 求和（3、5）。
输出 结果。
```

### 递归

函数体内嵌套块时，用 `。。` 结束函数体：

```
令 整数 阶 为（整数 n）：若 n 等于 1 则：返回 1。否则：返回 n 乘 阶（n 减 1）。。令 整数 结果 为 阶（5）。输出 结果。
```

内层 `。` 关闭 if 块，外层 `。` 关闭函数体。

## 错误处理

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| `变量未定义` | 使用未声明的变量 | 先用`令`声明 |
| `变量已存在` | 重复声明 | 使用`设`修改 |
| `除零错误` | 除数为零 | 检查除数 |
| `语法错误` | 格式不正确 | 检查 `：`/`。` 配对 |

## 最佳实践

```

```

### 注释

```
注：这个函数计算两个数的和
令 整数 求和 为（整数 甲、整数 乙）：返回 甲 加 乙。
```

### 错误处理

```
若 X 等于 0 则：输出 "除数不能为零"。否则：令 整数 结果 为 Y 除 X；输出 结果。
```

## 示例

### 九九乘法表

```
令 整数 甲 为 1。
当 甲 小于等于 9 时：
    令 整数 乙 为 1。
    当 乙 小于等于 甲 时：
        令 整数 积 为 甲 乘 乙。
        输出 积。
        设 乙 为 乙 加 1。
    设 甲 为 甲 加 1。
```

### 斐波那契数列

```
令 整数 斐 为（整数 n）：若 n 小于等于 1 则：返回 n。否则：返回 斐（n 减 1）加 斐（n 减 2）。。令 整数 I 为 0。当 I 小于 10 时：输出 斐（I）；设 I 为 I 加 1。
```
