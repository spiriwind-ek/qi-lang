# 奇语言 常见问题

## 基础问题

### Q: 如何安装奇语言？
A: 无需安装，直接运行 Python 脚本即可。确保已安装 Python 3.6+。

### Q: 如何运行代码？
A: 
```bash
python main.py your_code.qi
```

### Q: 如何进入交互式Shell？
A: 
```bash
python qish.py
```

### Q: 如何格式化代码？
A: 
```bash
python src/qifmt.py your_code.qi
```

## 语法问题

### Q: 为什么每条语句都要以句号结尾？
A: 句号是语句结束符，帮助解析器识别语句边界。

### Q: 为什么变量声明必须指定类型？
A: 奇语言是静态类型语言，类型在声明时确定，之后不能改变。

### Q: 数组索引为什么从1开始？
A: 这是设计选择，更符合中文用户的直觉（第1个、第2个...）。

### Q: 为什么 `令` 和 `设` 要分开？
A: `令` 用于声明新变量（只能用一次），`设` 用于修改已有变量的值。

## 运行时问题

### Q: 出现"变量未定义"错误怎么办？
A: 确保变量在使用前已用 `令` 声明。

### Q: 出现"变量已存在"错误怎么办？
A: 变量已声明后，使用 `设` 修改值，不要再次使用 `令`。

### Q: 出现"除零错误"怎么办？
A: 确保除数不为零。

### Q: 出现"语法错误"怎么办？
A: 检查：
- 每条语句是否以句号结尾
- 代码块是否用冒号开始
- 关键字拼写是否正确

## 函数问题

### Q: 函数可以递归调用吗？
A: 可以。函数可以调用自己。

### Q: 函数有返回类型吗？
A: 有。定义函数时必须指定返回类型。

```json
令 整数 求和 为（整数 甲、整数 乙）：
    返回 甲 加 乙。
```

### Q: 函数可以没有返回值吗？
A: 可以。使用 `空` 作为返回类型。

```
令 空 打印 为（文本 消息）：
    输出 消息。
```

## 数据结构问题

### Q: 列表和结构体有什么区别？
A: 
- 列表：存储相同类型的多个值
- 结构体：存储不同类型的多个值

### Q: 如何访问列表元素？
A: 使用1-based索引。

```
令 整数列 列表 为 [1、2、3]。
输出 列表[1]。    // 1
```

### Q: 如何访问结构体成员？
A: 使用 `的` 连接符。

```
令 结构 学生 含：文本 姓名。
令 学生 小明 为 学生含（"小明"）。
输出 小明的姓名。
```

## 模块问题

### Q: 如何导入其他文件？
A: 使用 `包括` 语句。

```
包括 "./数学工具.qi"。
```

### Q: 导入的文件中的变量在哪里可见？
A: 在当前文件的全局作用域内可见。

## 格式化问题

### Q: qifmt 做什么？
A: 将紧凑的代码格式化为标准格式，包括：
- 添加空格分隔关键字
- 添加冒号和缩进
- 归一化标点符号

### Q: 如何使用 qifmt？
A: 
```bash
python src/qifmt.py your_code.qi
```

### Q: qifmt 会修改原文件吗？
A: 不会。它只输出格式化后的代码到标准输出。
