## 冒泡排序

```
令 整数列 列表 为 [5、3、8、1、9]。

令 整数 I 为 0。
当 I 小于 4 时：
    令 整数 J 为 0。
    当 J 小于 4 减 I 时：
        若 列表[J] 大于 列表[J 加 1] 则：
            令 整数 临时 为 列表[J]。
            设 列表[J] 为 列表[J 加 1]。
            设 列表[J 加 1] 为 临时。
        设 J 为 J 加 1。
    设 I 为 I 加 1。

输出 列表[1]。    // 1
输出 列表[5]。    // 9
```

## 猜数字游戏

```
令 整数 目标 为 7。
令 整数 猜测 为 0。

当 猜测 不等于 目标 时：
    令 文本 输入 为 读取（"猜一个数字："）。
    设 猜测 为 输入 整数。
    若 猜测 大于 目标 则：
        输出 "太大了"。
    再若 猜测 小于 目标 则：
        输出 "太小了"。
    否则：
        输出 "恭喜你猜对了！"。
```
