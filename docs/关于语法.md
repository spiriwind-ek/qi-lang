# 奇语言 语法规范

## 设计哲学

用中文写代码，像写文章一样自然。

## 基本约定

1. 语句以 `。` 结尾
2. 代码块用 `：` 开始，`。` 结束（`：`/`。` 如括号配对）
3. 块内多条语句用 `；` 连接
4. 参数用 `、` 分隔
5. 变量声明必须指定类型，声明后类型不可变更
6. 成员访问用 `的`

## 标点符号

| 符号 | 用途 |
|------|------|
| `。` | 语句结束 / 块结束（匹配最近的 `：`） |
| `；` | 块内连接多条语句 |
| `：` | 代码块开始 |
| `、` | 分隔参数 |
| `（）` | 参数列表 |
| `注：` | 注释 |

## 类型系统

### 基础类型

| 类型 | 说明 |
|------|------|
| `整数` | 整数 |
| `小数` | 浮点数 |
| `文本` | 字符串 |
| `布尔` | 布尔值（真/假） |

| 类型 | 声明方式 | 说明 |
|------|----------|------|
| `整数列` | `令 整数列 X 为 [1、2、3]。` | 整数列表 |

| `结构` | `令 结构 学生 含：文本 姓名。` | 结构体 |

## 变量

### 声明

```
令 整数 X 为 10。
令 文本 名字 为 "张三"。
令 小数 PI 为 3.14。
令 真 标志 为真。
```

### 类型推导

不写类型，自动从值推断（静态类型，推断后不可变）：

```
令 X 为 10。         // 推断为整数
令 Y 为 3.14。       // 推断为小数
令 Z 为 "hello"。    // 推断为文本
令 B 为真。          // 推断为布尔
```

### 赋值

```
设 X 为 20。
设 X 为 X 加 1。
```

## 运算符

### 算术

| 运算 | 写法 | 示例 |
|------|------|------|
| 加 | `加` 或 `+` | `X 加 5` |
| 减 | `减` 或 `-` | `X 减 3` |
| 乘 | `乘` 或 `*` | `X 乘 2` |
| 除 | `除` 或 `/` | `X 除 4` |

### 比较

| 运算 | 写法 | 示例 |
|------|------|------|
| 等于 | `等于` 或 `==` | `X 等于 5` |
| 不等于 | `不等于` 或 `!=` | `X 不等于 0` |
| 大于 | `大于` 或 `>` | `X 大于 0` |
| 小于 | `小于` 或 `<` | `X 小于 10` |
| 大于等于 | `大于等于` 或 `>=` | `X 大于等于 0` |
| 小于等于 | `小于等于` 或 `<=` | `X 小于等于 10` |

### 逻辑

| 运算 | 写法 | 示例 |
|------|------|------|
| 与 | `且` 或 `&&` | `X 大于 0 且 X 小于 10` |

### 优先级

乘除 > 加减 > 比较 > 逻辑 > 赋值

## 控制结构

`：`/`。` 括号式配对——`：` 开块，`。` 关闭最近的 `：`。

### 条件

```
若 X 大于 0 则：输出 "正数"。否则：输出 "非正数"。
```

多语句块用 `；`：

```
若 X 大于 0 则：设 Y 为 1；设 Z 为 2。否则：设 Y 为 0。
```

多分支：

```
若 X 大于 0 则：输出 "正数"。再若 X 等于 0 则：输出 "零"。否则：输出 "负数"。
```

### 条件循环

```
当 X 小于 10 时：设 X 为 X 加 1。
```

多语句：

```
当 X 小于 3 时：设 X 为 X 加 1；输出 X。
```

### 计数循环

```
重复 5 次：输出 "Hello"。
```

### 遍历循环

```
令 整数列 水果 为 ["苹果"、"香蕉"、"橙子"]。
对 水果 每项 名称 于：输出 名称。
```

## 函数

### 定义

```
令 整数 求和 为（整数 甲、整数 乙）：返回 甲 加 乙。
```

多语句函数体：

```
令 整数 求和 为（整数 甲、整数 乙）：令 整数 结果 为 甲 加 乙；返回 结果。
```

### 递归

函数体内嵌套块时，用 `。。` 结束函数体：

```
令 整数 阶 为（整数 n）：若 n 等于 1 则：返回 1。否则：返回 n 乘 阶（n 减 1）。。令 整数 结果 为 阶（5）。输出 结果。
```

说明：内层 `。` 关闭 if 块，外层 `。` 关闭函数体。

### 调用

```
令 整数 结果 为 求和（3、5）。
输出 结果。
```

## 数组

### 声明与访问

```
令 整数列 库存 为 [10、20、30]。
输出 库存[1]。    // 10（1-based）
输出 库存[3]。    // 30
```


## 自然语言写法

代码可以写成一整行，也可以分多行。`qifmt` 格式化工具负责排版。

### 单行写法

```
令 整数 X 为 10。若 X 大于 0 则：输出 "正"。否则：输出 "负"。
```

### 分号连接

块内多条语句用 `；` 连接：

```
若 真 则：设 X 为 1；设 Y 为 2；输出 X 加 Y。
```

### 函数体

```
令 整数 双 为（整数 X）：令 整数 结果 为 X 乘 2；返回 结果。
```

### 嵌套块

函数体内有 if/while 等块时，用 `。。` 结束函数体：

```
令 整数 阶 为（整数 n）：若 n 等于 1 则：返回 1。否则：返回 n 乘 阶（n 减 1）。。令 整数 结果 为 阶（5）。输出 结果。
```

内层 `。` 关闭 if 块，外层 `。` 关闭函数体。

### 格式化后

`qifmt` 会自动整理成缩进格式：

```
令 整数 阶 为（整数 n）：
    若 n 等于 1 则：
        返回 1。
    否则：
        返回 n 乘 阶（n 减 1）。
令 整数 结果 为 阶（5）。
输出 结果。
```
