## 入门

### 第一个程序

创建文件 `hello.qi`：

```
注：这是我的第一个奇语言程序

```

运行：

```bash
python main.py hello.qi
```

```bash
python qish.py
```

## 运算

### 算术运算

```
令 整数 X 为 10。
令 整数 Y 为 3。

输出 X 加 Y。    // 13
输出 X 减 Y。    // 7
输出 X 乘 Y。    // 30
输出 X 除 Y。    // 3.333...
```

### 比较运算

```
令 整数 X 为 10。

输出 X 等于 10。      // 真
输出 X 不等于 5。     // 真
输出 X 大于 5。       // 真
输出 X 小于 20。      // 真
输出 X 大于等于 10。  // 真
输出 X 小于等于 10。  // 真
```

### 逻辑运算

```
令 真 A 为真。
令 真 B 为假。

输出 A 且 B。    // 假
输出 A 或 B。    // 真
输出 非 A。      // 假
输出 A 抑或 B。  // 真
```

### 运算符优先级

```
乘、除 > 加、减 > 比较 > 逻辑运算 > 赋值
```

示例：
```
令 整数 X 为 2 加 3 乘 4。   // X = 14，不是 20
令 整数 Y 为 (2 加 3) 乘 4。 // Y = 20
```

## 控制结构

### 条件语句

```
令 整数 X 为 10。

若 X 大于 0 则：
    输出 "正数"。
否则：
    输出 "非正数"。
```

### 多分支条件

```
令 整数 X 为 0。

若 X 大于 0 则：
    输出 "正数"。
再若 X 等于 0 则：
    输出 "零"。
否则：
    输出 "负数"。
```

### 条件循环

```
令 整数 X 为 0。
当 X 小于 10 时：
    设 X 为 X 加 1。
输出 X。    // 10
```

### 计数循环

```
重复 5 次：
    输出 "Hello"。
```

## 函数

### 定义函数

```
令 整数 求和 为（整数 甲、整数 乙）：
    返回 甲 加 乙。
```

### 调用函数

```
令 整数 结果 为 求和（3、5）。
输出 结果。    // 8
```

### 递归函数

```
令 整数 阶 为（整数 n）：
    若 n 等于 0 则：返回 1。
    返回 n 乘 阶（n 减 1）。

令 整数 结果 为 阶（5）。
输出 结果。    // 120
```

### 函数作为表达式

```
输出 求和（3、5）。    // 直接输出结果
```

## 错误处理

### 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| `变量未定义` | 使用未声明的变量 | 先用`令`声明变量 |
| `变量已存在` | 重复声明变量 | 使用`设`修改变量值 |
| `除零错误` | 除数为零 | 检查除数不为零 |
| `语法错误` | 代码格式不正确 | 检查句号、冒号 |
| `文件不存在` | 导入的文件路径错误 | 检查文件路径 |

### 错误信息格式

```
错误: 第X行第Y列: 错误描述
```

## 最佳实践

### 1. 命名规范

- 变量名：有意义的中文名称
- 函数名：动词开头，描述功能
- 常量：全大写

```
令 整数 用户年龄 为 18。
令 整数 计算平方 为（整数 X）：返回 X 乘 X。
常量 最大重试次数 为 3。
```

### 2. 代码组织

```
注：程序入口
令 整数 主函数 为（）：
    令 整数 结果 为 计算（）。
    输出 结果。

令 整数 计算 为（）：
    返回 42。
```

### 3. 注释

```
注：这个函数计算两个数的和
令 整数 求和 为（整数 甲、整数 乙）：
    返回 甲 加 乙。
```

### 4. 错误处理

```
若 X 等于 0 则：
    输出 "错误：除数不能为零"。
否则：
    令 整数 结果 为 Y 除 X。
    输出 结果。
```

### 5. 使用格式化工具

```bash
python src/qifmt.py your_code.qi
```

## 示例程序

### 计算器

```
令 整数 X 为 10。
令 整数 Y 为 5。

输出 "加法：" 加 X 加 Y。
输出 "减法：" 加 X 减 Y。
输出 "乘法：" 加 X 乘 Y。
输出 "除法：" 加 X 除 Y。
```

### 猜数字游戏

```
令 整数 目标 为 7。
令 整数 猜测 为 0。

当 猜测 不等于 目标 时：
    令 文本 输入 为 读取（"猜一个数字："）。
    设 猜测 为 输入 整数。
    若 猜测 大于 目标 则：
        输出 "太大了"。
    再若 猜测 小于 目标 则：
        输出 "太小了"。
    否则：
        输出 "恭喜你猜对了！"。
```

### 斐波那契数列

```
令 整数 斐 为（整数 n）：
    若 n 小于等于 1 则：返回 n。
    返回 斐（n 减 1）加 斐（n 减 2）。

令 整数 I 为 0。
当 I 小于 10 时：
    输出 斐（I）。
    设 I 为 I 加 1。
```
