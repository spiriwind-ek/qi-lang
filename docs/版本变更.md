# 版本变更记录

每个版本发布时总结，记录功能变更、修复、已知问题。

---

## v0.1.1-alpha（当前）

### 新增功能
- `：`/`。` 括号式配对块结构
- `；` 连接块内多条语句
- 三元表达式：`若 condition 则 A 否则 B`
- 一元负号：`-5`、`-(1+2)`
- 类型推导：`令 X 为 10` 自动推断
- 列表元素赋值：`设 列表[j] 为 值`
- 结构体字段修改：`设 小明的年龄 为 19`
- `::` 半角注释（归一化为 `注：`）
- 文件包含安全加固（路径沙箱）
- qish 管道模式和 readline 支持

### 重构
- 源码文件全部中文命名
- 配置迁移到 TOML
- qifmt 重写为 lexer 驱动，改名格式化奇.py
- 表达式解析器拆分为独立模块
- STMT_STARTERS 从配置导入
- 移除无用的 `=` 检测代码
- 测试文件全部中文命名
- 文档同步更新

### 已知问题
- 类型系统是装饰性的（不检查赋值类型）
- 测试覆盖有限（78 个测试，缺少边界情况）
- 语法分析器可进一步拆分
