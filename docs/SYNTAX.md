# 奇语言 语法规范

## 设计哲学

用中文实现程序的编写，像写文章一样写代码。

## 基本约定

1. 每条语句以句号 `。` 结尾
2. 代码块用冒号 `：` 开始，句号结束
3. 参数用顿号 `、` 分隔
4. 变量声明必须指定类型
5. 变量声明后类型不可变更

## 标点符号

| 符号 | 汉字 | 用途 |
|------|------|------|
| `。` | `。` | 语句结束 |
| `；` | `承` | 语句分隔 |
| `，` | `并` | 分隔参数 |
| `：` | `：` | 代码块开始 |
| `、` | `、` | 分隔参数 |
| `（）` | `（）` | 参数列表 |
| `注：` | `注：` | 注释 |

## 类型系统

### 基础类型

| 类型 | 说明 |
|------|------|
| `整数` | 整数 |
| `小数` | 浮点数 |
| `文本` | 字符串 |
| `真`/`假` | 布尔值 |
| `空` | 无类型 |
| `无` | 空值 |

### 复合类型

| 类型 | 声明方式 | 说明 |
|------|----------|------|
| `整数列` | `令 整数列 X 为 [1,2,3]。` | 整数列表 |
| `结构` | `令 结构 学生 含：文本 姓名。` | 结构体 |

## 变量

### 声明

```
令 整数 X 为 10。
令 文本 名字 为 "张三"。
令 小数 PI 为 3.14。
令 真 标志 为真。
```

### 赋值

```
设 X 为 20。
设 X 为 X 加 1。
```

### 类型转换

```
令 整数 X 为 "10" 整数。    // 字符串转整数
令 文本 Y 为 123 文本。      // 整数转字符串
```

## 运算符

### 算术运算

| 运算 | 汉字 | 示例 |
|------|------|------|
| 加 | `加` | `X 加 5` |
| 减 | `减` | `X 减 3` |
| 乘 | `乘` | `X 乘 2` |
| 除 | `除` | `X 除 4` |

### 比较运算

| 运算 | 汉字 | 示例 |
|------|------|------|
| 等于 | `等于` | `X 等于 5` |
| 不等于 | `不等于` | `X 不等于 0` |
| 大于 | `大于` | `X 大于 0` |
| 小于 | `小于` | `X 小于 10` |
| 大于等于 | `大于等于` | `X 大于等于 0` |
| 小于等于 | `小于等于` | `X 小于等于 10` |

### 逻辑运算

| 运算 | 汉字 | 示例 |
|------|------|------|
| 与 | `且` | `X 大于 0 且 X 小于 10` |
| 或 | `或` | `X 小于 0 或 X 大于 100` |
| 非 | `非` | `非 X 等于 0` |
| 异或 | `抑或` | `A 抑或 B` |

### 运算符优先级

```
乘、除 > 加、减 > 比较 > 逻辑运算 > 赋值
```

## 控制结构

### 条件

```
若 X 大于 0 则：
    输出 "正数"。
否则：
    输出 "非正数"。
```

### 多分支

```
若 X 大于 0 则：
    输出 "正数"。
再若 X 等于 0 则：
    输出 "零"。
否则：
    输出 "负数"。
```

### 条件循环

```
当 X 小于 10 时：
    设 X 为 X 加 1。
```

### 计数循环

```
重复 5 次：输出 "Hello"。
```

## 函数

### 定义

```
令 整数 求和 为（整数 甲、整数 乙）：
    返回 甲 加 乙。
```

### 调用

```
令 整数 结果 为 求和（3、5）。
输出 结果。    // 8
```

### 递归

```
令 整数 阶 为（整数 n）：
    若 n 等于 1 则：返回 1。
    返回 n 乘 阶（n 减 1）。
```

## 数组

### 声明

```
令 整数列 库存 为 [10、20、30]。
```

### 访问（1-based）

```
输出 库存[1]。    // 10
输出 库存[3]。    // 30
```

### 长度

```
输出 库存之长度。  // 3
```

## 结构体

### 定义

```
令 结构 学生 含：
    文本 姓名、
    整数 年龄。
```

### 实例化

```
令 学生 小明 为 学生含（"小明"、18）。
```

### 访问

```
输出 小明之姓名。    // "小明"
设 小明之年龄 为 19。
```

## 输入

```
令 文本 输入 为 读取（"请输入："）。
输出 输入。
```

## 运行时错误

发生以下情形时，解释器终止运行并输出错误位置与原因：

1. 算术除零
2. 变量重复声明或未定义
3. 缺少句号
4. 语法错误
