# 奇语言语法规范

## 设计哲学
用中文实现程序的编写，像写文章一样写代码。

- (a) 没有空格的写法是为了看上去更加连贯通顺。
- (b) 半角和全角的字符本质上是汉字的语法糖，使其更容易输入，qifmt格式化之后就会把它变回汉字（除了括号、中括号、冒号和句号，这些是元符号，没有对应的汉字）。

## 标点符号

| 符号 | 汉字 | 用途 |
|------|------|------|
| `。` | `。` | 语句结束 |
| `；` | `承` | 代码块内部语句分隔 |
| `，` | `并` | 分隔语句/参数 |
| `：` | `：` | 代码块开始/参数引入 |
| `、` | `、` | 分隔参数 |
| `（）` | `（）` | 全角括号，用于参数列表 |
| `注: ` | `注：` | 注释 |

## 关键字

| 关键字 | 用途 |
|--------|------|
| `令...为` | 声明变量（只能用一次） |
| `设...为` | 改变变量值 |
| `若...则...否则` | 条件 |
| `当...时` | 条件循环 |
| `重复...次` | 计数循环 |
| `返回` | 函数返回 |
| `读取` | 输入 |
| `输出` | 输出 |
| `真` | true |
| `假` | false |

## 运算符

| 半角 | 全角 | 汉字 | 用途 |
|------|------|------|------|
| `=` | `＝` | `为` | 赋值 |
| `==` | `＝＝` | `等于` | 比较 |
| `>` | `＞` | `大于` | 大于 |
| `<` | `＜` | `小于` | 小于 |
| `+` | `＋` | `加` | 值运算（数学加法、字符串连接） |
| `-` | `－` | `减` | 值运算（数学减法） |
| `*` | `×` | `乘` | 值运算（数学乘法） |
| `/` | `÷` | `除` | 值运算（数学除法） |
| `&&` | `＆＆` | `且` | 逻辑与 |
| `\|\|` | `｜｜` | `或` | 逻辑或 |
| `!` | `！` | `非` | 逻辑非 |
| `^` | `＾` | `抑或` | 逻辑异或 |
| `,` | `，` | `并` | 分隔语句/参数 |
| `;` | `；` | `承` | 代码块内部语句分隔 |

全角/半角/汉字都可以，不用统一。

### 运算符优先级
```
乘（×）、除（÷） > 加（+）、减（-） > 逻辑运算
```

### 逻辑运算符示例
```
若X>0且X<10则输出"在范围内"。
若X=1或X=2则输出"是1或2"。
若非X=0则输出"不是0"。
```

## 类型系统

静态类型：变量的类型在声明时确定，之后不能改变。

已知类型：
- `整数` — 整数
- `文本` — 字符串
- `真`/`假` — 布尔

### 类型转换
直接在值后面写目标类型名，不需要额外的关键字。

```
设X为"10"整数。   // 字符串 → 整数
设X为3.14整数。   // 小数 → 整数
令Y为123文本。     // 整数 → 字符串
```

## 变量

### 声明（令）
变量第一次出现时必须用`令`声明，声明后不能再次声明。
必须指定类型。

```
令整数X为1。       // 声明类型，赋值
令文本名字为"张三"。 // 声明，指定类型
```

### 改值（设）
变量声明后，只能用`设`改变其值，不能再次用`令`。
支持类型转换：直接在值后面写目标类型名。

```
令整数X为5。
设X为10。          // 正确
设X为X加1。        // 正确
设X为"10"整数。     // 正确，字符串转整数
令整数X为20。       // 错误！不能再次声明
```

### 规则
- `令` — 声明变量，只能用一次
- `设` — 改变变量值，可多次使用
- 类型在声明时确定，之后不能改变
- 变量名在同一作用域内不能重复

## 控制结构

### 条件
```
若X>0则输出"正数"，否则输出"非正数"。
```
- `若...则...否则` 是一个完整语句
- 内部用`，`分隔
- 整个语句用`。`结束

### 循环
```
当X<10时设X为X加1。
重复10次输出"你好"。
```

## 函数

```
令 整数 加法 为（整数一、整数二）：令 整数 结果 为 整数一 加 整数二 承 返回结果。
```

或者换行写法：
```
令 整数 加法 为（整数一、整数二）：
    令 整数 结果 为 整数一 加 整数二；
    返回 结果。
```

- 返回类型在前（整数）
- 函数名在后（加法）
- 参数列表用全角括号
- 用`：`开始函数体
- 用`返回`返回值
- 用`承`或`；`分隔语句
- 递归：函数允许在内部调用自己

## 数组

数组下标从1开始，不是0。

```
令整数数组为[1,2,3,4,5]。
输出数组[1]。  // 输出第一个：1
输出数组[5]。  // 输出最后一个：5
```

## 输入输出

```
令文本输入为读取（"请输入："）。
输出"你输入的是："加输入。
```

## 作用域

每个`：…。`代码块都创建一个新的作用域。

```
令整数X为5。
若X>0则：
    令整数Y为10。    // Y只能在这个代码块内使用
输出Y。           // 错误！Y已经出去了
```

**规则：**
- 在哪里声明，就在哪里能用
- 出了代码块就不能用了
- 函数参数和函数内变量都是局部的
- 不同代码块可以有同名变量
- 内部作用域不能用`令`声明与外部同名的变量
- 但可以用`设`修改外部变量的值，修改只在内部生效

## 示例

```
注：奇语言示例
令整数加法为（整数一、整数二）：
    令整数结果为整数一加整数二；
    返回结果。

令整数X为5。
令文本名字为"张三"。

若X>0则输出"正数"，否则输出"非正数"。

当X<10时设X为X加1。
重复10次输出"你好"。

结果=加法（3、5）。
输出结果。

令整数数组为[1,2,3,4,5]。
输出数组[1]。

令文本输入为读取（"请输入："）。
输出"你输入的是："加输入。
```

- 写法自由，像写字一样
- qifmt格式化工具
- 全角/半角/汉字符号兼容
- 数学符号保留
