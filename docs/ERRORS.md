# 错误信息参考

## 词法错误

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| `未知字符 'X'` | 遇到不认识的字符 | 检查是否使用了不支持的字符 |
| `未闭合的字符串` | 字符串没有结束引号 | 确保字符串用引号正确闭合 |

---

## 语法错误

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| `期望 TOKEN_X，得到 TOKEN_Y` | 语法结构不匹配 | 检查语句格式 |
| `不支持的语句: X` | 遇到不支持的语句类型 | 检查语法是否正确 |
| `意外的 token: X` | 遇到意外的token | 检查代码格式 |

---

## 运行时错误

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| `变量 'X' 未定义` | 使用未声明的变量 | 先用 `令` 声明变量 |
| `变量 'X' 已存在` | 重复声明变量 | 使用 `设` 修改变量值 |
| `算术除零` | 除数为零 | 确保除数不为零 |
| `循环超过最大迭代次数` | 疑似无限循环 | 检查循环条件 |
| `函数 'X' 未定义` | 调用未定义的函数 | 先定义函数 |
| `函数 'X' 期望 N 个参数` | 参数数量不匹配 | 检查参数数量 |
| `索引 X 越界` | 数组索引超出范围 | 检查索引值（1-based） |
| `文件 'X' 不存在` | 导入的文件不存在 | 检查文件路径 |

---

## 错误位置

错误信息格式：`第X行第Y列: 错误描述`

- `第X行`: 代码的行号
- `第Y列`: 代码的列号
- `错误描述`: 具体的错误信息

**示例：**
```
错误: 第1行第18列: 期望 DOT，得到 EOF
```

表示第1行第18列期望句号，但遇到了文件结束符。
