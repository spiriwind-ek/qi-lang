# CHANGELOG

## v0.1.2-alpha

### 新增功能
- **上下文关键字系统**：关键字可当变量名使用（`长度`、`文本库` 不再冲突）
- **字节码 VM**：栈式字节码虚拟机，支持所有核心语言特性（use_vm=True 开关）
- **指令集数据驱动**：指令定义在 `指令定义.toml` 中，与词法/语法定义风格一致
- **标准库**：基础库（数值化、文字化、取模、最大值、最小值）、文字库（长度、截取、查找、替换、转大小写、去留白）、列表库（排序、反转、去重、拥有、长度）、文件库（VFS 文件读写）
- `定义加载器.py`：配置拆分为词法/语法/运行时三个 TOML 文件

### 架构变更
- 词法分析器不再识别关键字，关键字判断推迟到语法分析器（`_resolve()` + `关键字.py`）
- 删除死代码 `表达式解析器.py`（mixin 从未被执行）
- 参数解析逻辑统一为 `_parse_args()`，消除 4 处代码重复
- 配置从单文件 `语言配置.toml` + `配置.py` 拆分为 `词法定义.toml` + `语法定义.toml` + `运行时定义.toml` + `定义加载器.py`

### 新增文件
- `src/指令定义.toml` — 指令集定义
- `src/指令加载器.py` — 指令加载器
- `src/字节码.py` — Chunk、ObjFunction 等数据结构
- `src/编译器.py` — AST→字节码 编译器
- `src/虚拟机.py` — 栈式字节码执行引擎
- `src/关键字.py` — 关键字分类映射表
- `src/词法定义.toml` / `src/语法定义.toml` / `src/运行时定义.toml`
- `src/定义加载器.py` — 配置加载器
- `src/标准库/` — 4 个标准库模块
- `tests/虚拟机测试.py` — 22 个 VM 测试
- `tests/标准库测试.py` — 18 个标准库测试

### 删除文件
- `src/表达式解析器.py`（死代码）
- `src/配置.py`（拆分为定义加载器）
- `src/语言配置.toml`（拆分为三个 TOML）
- `src/标准库/文本库.py`（被 `文字库.py` 替代）

### 修复
- 上下文关键字系统：`令狐冲` 不再被拆为 `令(LET)` + `狐冲`
- `_compile_foreach` 循环条件逻辑修复
- 词法分析器测试适配上下文关键字系统

## v0.0.1-alpha (2026-07-18)

### 新增功能
- 变量声明（令）和赋值（设）
- 整数、小数、文本、布尔类型
- 输出语句
- 用户输入（读取）
- 算术运算（加减乘除）
- 比较运算（等于、不等于、大于、小于等）
- 逻辑运算（且、或、非、抑或）
- 条件分支（若...则...否则）
- 多分支条件（再若）
- 条件循环（当...时）
- 计数循环（重复...次）
- 函数定义和调用
- 递归支持
- 列表（1-based索引）
- 结构体定义和访问
- 模块导入（包括）
- 注释（注：）
- 交互式Shell (qish)
- qifmt代码格式化工具

### 修复
- 词法分析器：全角标点归一化
- 语法分析器：函数体支持多语句
- 解释器：递归函数调用
- qifmt：输出语句不加冒号
