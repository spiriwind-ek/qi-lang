# CHANGELOG

## v0.0.1-alpha (2026-07-18)

### 新增功能
- 变量声明（令）和赋值（设）
- 整数、小数、文本、布尔类型
- 输出语句
- 用户输入（读取）
- 算术运算（加减乘除）
- 比较运算（等于、不等于、大于、小于等）
- 逻辑运算（且、或、非、抑或）
- 条件分支（若...则...否则）
- 多分支条件（再若）
- 条件循环（当...时）
- 计数循环（重复...次）
- 函数定义和调用
- 递归支持
- 列表（1-based索引）
- 结构体定义和访问
- 模块导入（包括）
- 注释（注：）
- 交互式Shell (qish)
- qifmt代码格式化工具

### 修复
- 词法分析器：全角标点归一化
- 语法分析器：函数体支持多语句
- 解释器：递归函数调用
- qifmt：输出语句不加冒号
